# 🚀 Profile Card Generator

A modern and responsive **Profile Card Generator** built with **HTML, CSS, and JavaScript**.

## ✨ Features

* 📷 Upload Profile Image
* ⚡ Live Preview
* 🎨 Custom Accent Color
* 🌙 Dark Mode
* 🛠 Skills & Social Links
* 📥 Download as PNG
* 📄 Download as PDF
* 💾 Auto Save (Local Storage)
* 📱 Responsive Design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Font Awesome

## 🚀 Getting Started

1. Clone the repository.
2. Open `index.html` in your browser or use **Live Server**.
